# Fraction Calculator

A lightweight, SEO-friendly microtool built with Next.js App Router and TypeScript. It helps users add, subtract, multiply, divide, and simplify fractions while also showing mixed-number and decimal output.

## Why this tool

"Fraction calculator" is a durable, high-intent search topic with broad demand from students, parents, teachers, cooks, and DIY users.

## Core features

- Add, subtract, multiply, and divide fractions
- Auto-simplified exact answers, mixed numbers, and decimal output
- Accessible, static-first marketing pages with sitemap and robots support

## Scope cuts

- No saved history or user accounts
- No step-by-step algebra explanation engine
- No ad integrations yet

## Tech stack

- Next.js App Router
- TypeScript
- React 19
- Plain CSS

## Local development

```bash
npm install
npm run dev
```

## Production notes

Set `NEXT_PUBLIC_SITE_URL` in your deployment environment so canonical tags, sitemap, and robots output use the final domain.
